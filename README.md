## Descrição

Este projeto implementa uma API para gerenciar tarefas (TODOs), seguindo o tutorial oficial da Microsoft para [criar a primeira API web com ASP.NET Core](https://learn.microsoft.com/pt-br/aspnet/core/tutorials/first-web-api?view=aspnetcore-9.0&tabs=visual-studio-code).

A API permite realizar operações básicas de CRUD (Create, Read, Update, Delete) em uma lista de tarefas.

## Funcionalidades

- **Adicionar uma tarefa**
- **Listar todas as tarefas**
- **Consultar uma tarefa específica por ID**
- **Atualizar uma tarefa existente**
- **Excluir uma tarefa**

## Tecnologias Utilizadas

- **ASP.NET Core 9.0** - Framework principal para construção da API.
- **Entity Framework Core** - Para persistência de dados em memória ou em banco de dados.
- **C#** - Linguagem de programação principal.

## Pré-requisitos

Antes de começar, certifique-se de que você tenha o seguinte instalado em sua máquina:

- [.NET SDK 9.0 ou superior](https://dotnet.microsoft.com/download)
- [Visual Studio Code](https://code.visualstudio.com/) (ou qualquer IDE de sua preferência)
